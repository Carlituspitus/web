¿Qué es un sistema operativo?

  1. El código que acompaña la ejecución de cualquier aplicación.

  2. Un programa que actúa como intermediario entre un usuario y el hardware del computador.

  3. Un programa que dirige todos los recursos de un sistema computacional.


¿Cuáles son sus funciones?

  1. Acercar el sistema computacional al usuario facilitando su uso.

  2. Usar el hardware de una manera eficiente.

  3. La mejor forma de entender un S.O. es entender como se construye.

  4. También ayuda a su comprensión el tener una visión histórica de su desarrollo.


Estructura del sistema computacional

Un Sistema Computacional está compuesto de: hardware, s.o., aplicaciones

  1. Hardware- Proveé los componentes básicos de cómputo (CPU, memoria, dispositivos de E/S).

  2. Sistema Operativo- Controla y coordina el uso del hardware entre los varios programas de aplicación para los diferentes usuarios.

  3. Programas de sistemas- Compiladores, Assembler, Editores, herramientas de Monitoreo y Mantención.

  4. Aplicaciones - Bases de datos, Juegos, Navegadores, etc.


Tipos de sistemas operativos

Según el número de usuarios:

  1. Según el número de usuarios:

      2. Monousuario y multiusuario

  2. Según el uso de recursos:

      3. Monotarea y multitarea

  3. Según el número de microprocesadores:

      4. Multiprocesamiento o distribuidos

  4. Según el tiempo de respuesta:

      Procesamiento por lotes o de tiempo compartido

  5. Según la función que realizan:

      6. De red y cliente


COMPONENTES DEL SISTEMA OPERATIVO

  1. Manejador de recursos-

      2. administra y aloja los recursos.

  2. Programa de control-

      3. controla la ejecución de los programas de usuarios y las operaciones de los dispositivos de entrada/salida.

  3. Kernel-

      4. el programa que corre en todo momento (todo lo demas es programa de aplicación).

Necesitamos el sistema operativo por que prove interfaces de usuario, interprete de comandos, estructura de directorios, programas de utilerias (compiladores, editores, filtros).



UNIDAD 2 Internet

El origen del internet se remonta a la década de 1960, cuando el Departamento de Defensa de los Estados Unidos desarrolló la red ARPANET, que fue el precursor de lo que hoy conocemos como internet.

¿Quien lo inventó?

Originalmente, el Internet fue inventado por DARPA cuyas siglas en español significan Agencia de Proyectos de Investigación Avanzada para la Defensa, una organización estadounidense que buscaba un medio, durante la Guerra Fría.

¿Qué es el internet?

Internet se podría definir como una red global de redes de ordenadores cuya finalidad es permitir el intercambio libre de información entre todos sus usuarios. Pero sería un error considerar Internet únicamente como una red de computadoras.

¿Qué es una tarjeta de red?

Una tarjeta de red (también llamada placa de red o Network Interface Card (NIC)) es una clase de tarjeta destinada a ser introducida en la placa madre de una computadora o que se conecta a uno de sus puertos para posibilitar que la máquina se sume a una red y pueda compartir sus recursos (como los documentos, la conexión a Internet o una impresora, por ejemplo).

¿Qué es y cuál es la función de un router?

Un router es un dispositivo de hardware que sirve de punto de conexión entre una red local e Internet.

¿Qué es un HTML ?

HTML son las siglas en inglés de HyperText Markup Lenguage, que significa Lenguaje de Marcado de HiperTexto. Se llama así al lenguaje de programación empleado en la elaboración de páginas Web, y que sirve como estándar de referencia para la codificación y estructuración de las mismas, a través de un código del mismo nombre (html).

Servicios de internet

  1. Almacenamiento en la nube:

    1. Permite guardar información en internet.
    2. Se accede desde cualquier dispositivo con internet.
    3. Algunos ejemplos son Dropbox, Google Drive y OneDrive.

  2. Foros de internet:

    1. Espacios donde las personas conversan sobre temas de interés.
    2. Se publican y responden mensajes.
    3. Se crea un sentido de comunidad virtual.

  3. Redes sociales:

    1. Plataformas que permiten la interacción entre personas y empresas.
    2. Se comparte información, ideas y opiniones.
    3. Algunos ejemplos son Facebook, Twitter e Instagram.

  4. Plataformas de aprendizaje:

    1. Sitios web diseñados para facilitar el aprendizaje.
    2. Ofrecen cursos, materiales y herramientas online.
    3. Algunos ejemplos son Moodle, Edmodo y Khan Academy.

  5. Seguridad:

    1. Conjunto de medidas para proteger la información y equipos en internet.
    2. Se buscan y eliminan vulnerabilidades.
    3. Se crean métodos para proteger la red.

Tipos de redes informáticas

  1. LAN:

      2. Conecta dispositivos en un área limitada (hogar, oficina, edificio). Ejemplos: computadoras, impresoras, routers.

  2. MAN:

      3. Conecta redes LAN en un área metropolitana (ciudad, pueblos).

  3. WAN:

      4. Conecta redes a grandes distancias (países, continentes). Usos: comunicación, recursos, información.

  4. CAN:

      5. Conecta redes LAN en un campus educativo o corporativo

  5. SAN:

      6. Conecta dispositivos de almacenamiento a servidores. Alta velocidad, independiente de LAN/WAN.

  6. VLAN:

      7. Agrupa dispositivos en una LAN por tráfico, no por ubicación física.

  7. PAN:

      8. Conecta dispositivos electrónicos en un área reducida (teléfono, impresora, etc.).

NAVEGADORES WEB

Los navegadores web son aplicaciones que permiten a los usuarios acceder y navegar por páginas web en Internet. Algunos de los navegadores más populares incluyen Google Chrome, mozilla Firefox, Microsoft edge y safari.

Virus y antivirus

Un virus informático es un tipo de programa de computadora malicioso que, así como los virus biológicos, se auto-replica y se esparce por el sistema infectando distintas partes del mismo. Además, está diseñado para propagarse de un dispositivo a otro.

¿Qué hace el antivirus?

  1. Proteger tu equipo de virus y malware.
  2. Proviene la perdida de datos.
  3. Ofrece seguridad en línea.
  4. Bloquea el uso indebido de tus recursos.
  5. Adaptado a las últimas amenazas de ciberseguridad.

UNIDAD 5 Funciones trigonométricas

Introducción

Las funciones trigonométricas son relaciones matemáticas que se basan en los ángulos de un triángulo rectángulo. Permiten relacionar las longitudes de sus lados y calcular medidas desconocidas.



Las funciones trigonométricas principales son:

  1. Seno (sin):

      2. Relación entre el cateto opuesto al ángulo y la hipotenusa.

  2. Coseno (cos):

      3. Relación entre el cateto adyacente al ángulo y la hipotenusa.

  3. Tangente (tan):

      4. Relación entre el cateto opuesto al ángulo y el cateto adyacente.



¿Cuáles son las principales?

  1. Seno (sin):

      2. Cateto opuesto / Hipotenusa.

  2. Coseno (cos):

      3. Cateto adyacente / Hipotenusa.

  3. Tangente (tan):

      4. Cateto opuesto / Cateto adyacente.


¿Para qué sirven?

  1. Calcular distancias, ángulos y otras magnitudes.
  2. Modelar fenómenos como el movimiento de los astros o las ondas.
  3. Crear gráficos y animaciones.
